# I-excel
Takes a screenshot of specific point and asks AI about it

v1.06

# Setup
# Download Ahk
Enter the link https://www.autohotkey.com/download/ahk-v2.exe

Open the file, in case you don't have admin access select current user and then do the following settups
## Chrome
Bookmarks bar open, it can be opened by pressing Ctrl+Shift+B

Make sure that chrome is the default browser, to assure it is go to chrome://settings/defaultBrowser.

## Snipping Tool
After that, go to settings>notifications then turn off Snipping Tool notifications.

Open the snipping tool by pressing Ctrl+E
## Windows settings
Make sure the color is set to blue

Color can be changed at: Personalization>Colors

Hover over the colors until you get blue
